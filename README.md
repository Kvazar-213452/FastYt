# Next.js Video Downloader with Python Microservice

## Overview

This project is a pet project created to practice modern full-stack web development using Next.js and a Python-based microservice architecture.

The application allows users to download videos from YouTube and other supported platforms by providing a video URL. The frontend is built with Next.js, while the backend is implemented as a Python microservice that handles video downloading using the yt-dlp library.

---

## Architecture

- Frontend: Next.js application
- Backend: Python microservice
- Communication: HTTP API

---

## Features

- Web interface built with Next.js
- Python microservice backend
- Video downloading from YouTube and other platforms
- Support for multiple formats and quality options
- Clean separation between frontend and backend
- Designed as a learning-oriented pet project

---

## Technologies Used

### Frontend
- Next.js
- React
- TypeScript

### Backend
- Python
- Flask or FastAPI
- yt-dlp
